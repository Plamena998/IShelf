# IShelf
iShelf is a minimalist, interactive personal library manager designed with an Apple-style aesthetic and a focus on tactile user experience. It transforms a digital book list into a physical-feeling collection with 3D animations and dynamic shelving.

✨ Key Features
Smart Virtual Shelves: Automatically organizes books into rows of 7, creating a new wooden shelf as your collection grows.

Tactile 3D Books: Hover over any book to see a realistic 3D "opening" animation with multiple pages.

Live Search & API: Powered by the OpenLibrary API with a debounced live search for high performance.

Handwritten Notes: A custom-built "Sticky Note" in the header with handwriting fonts and a persistent archive for your reading thoughts.

Progress Tracking: Visual stats bar showing total books, books read, and your overall percentage completion.

Data Persistence: All data, books, and notes are saved locally in your browser via localStorage.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Advanced 3D transforms & Glassmorphism)

Logic: Vanilla JavaScript (ES6+)

Data: OpenLibrary Search API

Fonts: Inter & Caveat (Google Fonts)
